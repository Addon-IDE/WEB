JSON 教程

JSON: JavaScript Object Notation（JavaScript 对象表示法）

JSON 是存储和交换文本信息的语法。

JSON 相对更小、更快，更易解析。

JSON 易于人阅读和编写。

众多编程语言都支持 JSON。

总之，JSON 是一种轻量级的数据交换格式。

接下来，我们将学习 JSON。

我会使用非常简单的方式来解释它，希望这对你有所帮助。

有链接样式的内容在上方/下方，点击即可跳转。

如果你想学习更多，可以进一步学习，这是非必要的。

（聪明的你发现了吗？进一步学习是橙色的！）

在学习之前，我该做什么？

适当了解一些基础的概念：

  • JSON 可以是有格式的，可能有很多空白。
  • 以下提及的字符均为半角(英文)字符。

JSON 值

JSON 值可以是：

  • 数字
  • 字符串
  • 逻辑值
  • 空值
  • 数组
  • 对象

书写 JSON 数字

  1. 数字可以是整数或小数(浮点数)。
  2. 小数的小数点前后至少有一位数字。
  3. 非负数的正号可有可无。
  4. 可以用类似科学记数法表示。

下面是数字的几个例子：

123
-1234.567
+0.55

书写 JSON 字符串

  1. 按照标准，必须使用双引号( "" )括起来。
  2. 字符串可以是任何内容，特殊的需要转义。
  3. 反斜杠( \ ) 是转义字符。

下面是字符串的几个例子：

"Hello World!"
""
"114514"

书写 JSON 逻辑值

  1. 逻辑值只有 真( true ) 和 假( false )。

下面是逻辑值的唯二写法：

true
false

书写 JSON 空值

  1. 空值是一个特殊的值，表示空对象引用。

下面是空值的唯一写法：

null

书写 JSON 数组

  1. 数组是一组由逗号分隔的 JSON 值。
  2. 数组必须用中括号( [] ) 括起来。

下面是数组的一个例子：

[
    "蔡徐坤",
    true,
    1998,
    "浙江省温州市"
]

书写 JSON 对象

  1. 对象由多个键值对组成，用逗号隔开。
  2. 对象必须用大括号( {} ) 括起来。

下面是对象的一个例子：

{
    "姓名": "蔡徐坤",
    "男性": true,
    "出生年份": 1998,
    "出生地区": "浙江省温州市"
}

键值对？那是什么？

  1. 键值对由一个键和一个值组成。
  2. 键与值之间使用冒号分隔。
  3. 按照标准，键是字符串，值的类型不限。

下面是键值对的一个例子：

"键": "值"

数字：科学记数法

科学记数法通常用于表示很大或很小的数字。

我们先要了解，在数学中的科学记数法应该怎么写。

科学计数法：a × 10n（1 ≤ |a| < 10，n 是整数）[1]

例如，1400000000 = 1.4 × 109 。

再者，0.00000012 = 1.2 × 10-7 。

好的，让我们来学学 JSON 中的科学记数法吧。

a × 10n = a + "e" + n（若 n 为正数，正号可有可无）

其实做个替换就好啦，并且 a 也符合数字的解释！

比如， -7e8 ，等于 -7 × 108 。

下面是科学记数法的几个例子：

1.4e9
1.2e-7
+0.12e-6
-9e+16
1. 在 JSON 中，a 的值可以在区间之外。↑

字符串：转义

  1. 反斜杠( \ ) 是转义符。
  2. 只有下列字符是有效的。
  3. 转义符要放在字符前面。

被转义后的有效字符：(都加上了转义符)

\"
\\
\/
\b
\f
\n
\r
\t
\u

它们所对应的解释大概是：

\"  - 在字符串中写双引号的方式
\\  - 写反斜杠的方式（因为自身是转义符）
\/  - 转不转义都可以（为兼容特殊情况）
\b  - 退格，将当前位置移到前一列
\f  - 换页，将当前位置移到下页开头
\n  - 换行，将当前位置移到下一行开头
\r  - 回车，将当前位置移到本行开头
\t  - 水平制表，将当前位置移到下一制表符
\u  - Unicode 码，后跟四位 16 进制字符

下面是含转义字符的字符串的一个例子：

"换行符：\"\\n\"\n王境泽曾说：真\u9999。"

按照标准的 JSON，会被认为是：

换行符："\n"
王境泽曾说：真香。
（\u9999 中的 9999 是 16 进制 Unicode 码，不重要）
