JSON 教程

JSON: JavaScript Object Notation（JavaScript 物件表示法）

JSON 是儲存和交換文字資訊的語法。

JSON 相對更小、更快，更易解析。

JSON 易於人閱讀和編寫。

眾多程式語言都支援 JSON。

總之，JSON 是一種輕量級的資料交換格式。

接下來，我們將學習 JSON。

我會使用非常簡單的方式來解釋它，希望這對你有所幫助。

有連結樣式的內容在上方/下方，點選即可跳轉。

如果你想學習更多，可以進一步學習，這是非必要的。

（聰明的你發現了嗎？進一步學習是橙色的！）

在學習之前，我該做什麼？

適當瞭解一些基礎的概念：

  • JSON 可以是有格式的，可能有很多空白。
  • 以下提及的字元均為半形(英文)字元。

JSON 值

JSON 值可以是：

  • 數字
  • 字串
  • 邏輯值
  • 空值
  • 陣列
  • 物件

書寫 JSON 數字

  1. 數字可以是整數或小數(浮點數)。
  2. 小數的小數點前後至少有一位數字。
  3. 非負數的正號可有可無。
  4. 可以用類似科學記數法表示。

下面是數字的幾個例子：

123
-1234.567
+0.55

書寫 JSON 字串

  1. 按照標準，必須使用雙引號( "" )括起來。
  2. 字串可以是任何內容，特殊的需要轉義。
  3. 反斜槓( \ ) 是跳脫字元。

下面是字串的幾個例子：

"Hello World!"
""
"114514"

書寫 JSON 邏輯值

  1. 邏輯值只有 真( true ) 和 假( false )。

下面是邏輯值的唯二寫法：

true
false

書寫 JSON 空值

  1. 空值是一個特殊的值，表示空物件引用。

下面是空值的唯一寫法：

null

書寫 JSON 陣列

  1. 陣列是一組由逗號分隔的 JSON 值。
  2. 陣列必須用中括號( [] ) 括起來。

下面是陣列的一個例子：

[
    "蔡徐坤",
    true,
    1998,
    "浙江省溫州市"
]

書寫 JSON 物件

  1. 物件由多個鍵值對組成，用逗號隔開。
  2. 物件必須用大括號( {} ) 括起來。

下面是物件的一個例子：

{
    "姓名": "蔡徐坤",
    "男性": true,
    "出生年份": 1998,
    "出生地區": "浙江省溫州市"
}

鍵值對？那是什麼？

  1. 鍵值對由一個鍵和一個值組成。
  2. 鍵與值之間使用冒號分隔。
  3. 按照標準，鍵是字串，值的型別不限。

下面是鍵值對的一個例子：

"鍵": "值"

進一步學習

格式化：空白

JSON 可以是有格式的，可能有很多空白。

空白可以是以下符號：

空格
換行符
回車符
水平製表符

數字：科學記數法

科學記數法通常用於表示很大或很小的數字。

我們先要了解，在數學中的科學記數法應該怎麼寫。

科學計數法：a × 10n（1 ≤ |a| < 10，n 是整數）[1]

例如，1400000000 = 1.4 × 109 。

再者，0.00000012 = 1.2 × 10-7 。

好的，讓我們來學學 JSON 中的科學記數法吧。

a × 10n = a + "e" + n（若 n 為正數，正號可有可無）

其實做個替換就好啦，並且 a 也符合數字的解釋！

比如， -7e8 ，等於 -7 × 108 。

下面是科學記數法的幾個例子：

1.4e9
1.2e-7
+0.12e-6
-9e+16
1. 在 JSON 中，a 的值可以在區間之外。↑

字串：轉義

  1. 反斜槓( \ ) 是轉義符。
  2. 只有下列字元是有效的。
  3. 轉義符要放在字元前面。

被轉義後的有效字元：(都加上了轉義符)

\"
\\
\/
\b
\f
\n
\r
\t
\u

它們所對應的解釋大概是：

\"  - 在字串中寫雙引號的方式
\\  - 寫反斜槓的方式（因為自身是轉義符）
\/  - 轉不轉義都可以（為相容特殊情況）
\b  - 退格，將當前位置移到前一列
\f  - 換頁，將當前位置移到下頁開頭
\n  - 換行，將當前位置移到下一行開頭
\r  - 回車，將當前位置移到本行開頭
\t  - 水平製表，將當前位置移到下一製表符
\u  - Unicode 碼，後跟四位 16 進位制字元

下面是含跳脫字元的字串的一個例子：

"換行符：\"\\n\"\n王境澤曾說：真\u9999。"

按照標準的 JSON，會被認為是：

換行符："\n"
王境澤曾說：真香。
（\u9999 中的 9999 是 16 進位制 Unicode 碼，不重要）
